## Definition

- An agent is a system that leverages an AI model to interact with its environment in order to achieve a user-defined objective
- It combines reasoning, planning, and the execution of actions (often via external tools) to fulfil tasks

## Components

### Brain

- AI model
- Handles reasoning and planning
- Decides which actions to take based on the situation

### Body

- Capabilities and tools
- Represents everything the agent is equipped to do
- The scope of possible actions depends on what the agent has been equipped with
